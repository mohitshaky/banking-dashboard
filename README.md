## Key Results
- ✅ Loads in under 1 second
- ✅ Handles 1,000+ transaction display
- ✅ Zero dependencies

## Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · REST API integration

## What It Does
A lightweight banking dashboard frontend that renders account summaries, balances, and paginated transaction history. Built with zero npm dependencies — pure HTML, CSS, and JavaScript — it integrates directly with the banking-account-service REST API and is deployable as a static site to any CDN or web server.

## Quick Start
```bash
# clone and run
git clone https://github.com/mohitshaky/banking-dashboard.git
cd banking-dashboard
./gradlew bootRun
```
